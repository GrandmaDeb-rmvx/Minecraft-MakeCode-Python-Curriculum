✨ Teaching Philosophy
My curriculum is designed to:
- Build confidence through hands-on, approachable problem-solving
- Foster joy through engaging environments (Minecraft, MakeCode, Python)
- Provide strong support for teachers who are new to Python
- Develop computational thinking, creativity, user-centered design skills, and resilience
- Support both new CS learners and students with prior experience
