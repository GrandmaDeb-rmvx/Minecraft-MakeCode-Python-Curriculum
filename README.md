 
🧱 Minecraft MakeCode Python Curriculum Repository

Welcome! This repository contains my computer science curriculum built around Minecraft MakeCode Python for middle and high school students. The lessons are sequential, teacher‑friendly, and designed to build Python skills through hands‑on projects, debugging practice, and meaningful assessments.

🚧 Curriculum Status
This curriculum is under active construction. The CS1 course is organized by instructional units, not by academic quarters.
The initial public release includes Unit 00, Unit 01, and Unit 02, with additional units added over time.
 


### ✔️ Phase 1 — CS1: Curriculum (In Progress)

Phase 1 represents the initial public release of the CS1 curriculum.
This phase focuses on helping students move from setup to confidently
executing and modifying programs in an interactive world.

By the end of Unit 02, students can:
- Set up their tools and workflow
- Read and run existing programs
- Understand how code executes step by step
- Modify program behavior through commands and parameters
- Interact with the world through code (actions, movement, and effects)
- Reason about cause-and-effect between code and in‑game outcomes

**Included in this phase:**
- **Unit 00 — Get Set Up**  
  Environment setup, expectations, and orientation to tools and workflow

- **Unit 01 — Hello World**  
  Running first programs, understanding structure, and making small changes. Students use user‑facing design to create clear, expressive output while developing foundational debugging skills and good programming practice.

- **Unit 02 — Execute**  
  Using execution commands in a variety of contexts, including world
  interactions such as entities, effects, environment checks, and agent actions

*Additional CS1 units will expand on these foundations as they are completed.*

✨ Teaching Philosophy
My curriculum is designed to:
- Build confidence through hands-on, approachable problem-solving
- Foster joy through engaging environments (Minecraft, MakeCode, Python)
- Provide strong support for teachers who are new to Python
- Develop computational thinking, creativity, user-centered design skills, and resilience
- Support both new CS learners and students with prior experience
