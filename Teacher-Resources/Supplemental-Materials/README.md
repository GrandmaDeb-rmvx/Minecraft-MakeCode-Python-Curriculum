# Supplemental Materials

This folder contains optional and supporting materials used across lessons to
reinforce familiarity with Minecraft terminology and concepts.

Materials here are typically used for:
- Vocabulary reinforcement (e.g., mob names, game elements)
- Warm‑ups or early‑finisher activities
- Review or confidence‑building practice
- Flexible support for students with varied Minecraft experience

These resources are **not required for core lessons** and may be used selectively
based on class needs.

## Current Status
This folder is a growing collection. Not all supplemental materials are uploaded yet.

At present, one representative document is included to establish structure and
provide an example of how these materials are used. Additional resources may be
added over time.

## Teacher Note
All materials in this folder are intended for teacher use and discretion.
