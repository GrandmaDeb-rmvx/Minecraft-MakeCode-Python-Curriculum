## Formative Assessments

These resources are designed to check student understanding, confidence,
or familiarity during instruction. They can function as exit tickets. They can be graded or ungraded.

Some formative assessments include a lesson name. These assessments can be used at the end of these lessons, or after.

"Skills" assessments are not intended to be graded.
