# Unit 00: Before You Begin

## 📘 Unit Overview
Unit 00 prepares students to access all required platforms and tools before beginning their coding journey. Students learn essential classroom and digital routines that will support their success throughout the CS1 course. This unit focuses on account setup, navigation skills, and establishing consistent habits for working in Minecraft Education and Minecraft.MakeCode.com.

---

## 🧰 Teacher Preparation
- Have student credentials available for:  
  - school email  
  - Office.com  
  - Minecraft Education  
- Ensure Minecraft Education is installed and updated on student devices.
- Confirm network access to:  
  - https://minecraft.makecode.com  

---

## 🧒 Student Skills (SWBAT)
- Sign into a school computer  
- Sign into school email (online)  
- Sign into Office.com (online)  
- Sign into Minecraft.MakeCode.com (online)  
- Create a project in MakeCode using the **green plus sign**  
- Identify colorful drawer categories and the coding workspace  
- Sign into Minecraft Education  
- Name a Minecraft world  
- Open the coding window using **C** on the keyboard  
- Select **MakeCode** (pink puzzle piece) as the coding platform  
- Create a new coding project using the **Python Only** option  

---

## 📦 Materials Needed
- School email login credentials  
- Office.com credentials  
- Access to Minecraft.MakeCode.com  
- Student devices with Minecraft Education installed
