# Unit 00: Before You Begin

> **Audience:** Teacher‑facing (TF). This README is for instructors. Student handouts live in **Teacher-Resources/** with the **-SF** suffix and are linked below.

---

## 📘 Unit Overview
Unit 00 prepares students to access all required platforms and tools before coding. Students establish classroom routines (naming projects, world settings) so Unit 01 can start smoothly.

---

## 🧰 Teacher Preparation
- Have credentials available for **school email**, **Office.com**, and **Minecraft Education**.
- Verify access to **Minecraft Education** on devices and to **https://minecraft.makecode.com**.
- (Optional) Prepare for a variety of levels of success for logins by having unplugged activities available.
- (Optional) Print or post the Student‑Facing handout(s) linked below.

---

## 🧒 SWBAT (Student Skills)
Students will be able to:
- Sign into a school computer.
- Sign into **school email** (online).
- Sign into **Office.com** (online).
- Sign into **Minecraft.MakeCode.com** (online).
- Create a project in MakeCode using the **green plus sign**  
- Choose a name for a project (using classroom standards)
- Choose python only setting
- Identify colorful drawer categories and the coding workspace  
- Identify the **Run** and **Share** buttons.
- Locate the blue player drawer, and drag out the player.say command

## 📝 Teacher Notes
- Keep Unit 00 **short**. The goal is technical readiness, not content coverage.
- Establish **naming conventions** (e.g., `U01L03-Firstname-Lastname`).
- Give a **tiny win** on day one (e.g., drag out the player.say command`).
- If students land in **Blocks/JS**, switch to **Python** and delete leftover code.
