
# U01L02 — Hello Coding
Timing
Early in Unit 01, pacing varies widely based on student confidence, typing skills, and technical readiness.
Plan for 30–45 minutes, but treat the lesson as complete once students demonstrate the target skills below.
Focus Skills
This lesson emphasizes workflow and reuse, rather than new syntax.
Students practice:

reopening an existing coding project
copying and pasting working code to avoid errors
creating multiple lines of output
copying and sending code via email using classroom conventions

Evidence of Completion
Collect evidence that students can:

run a Python program with four sequential player.say() outputs
reuse copied code successfully (no syntax errors)
submit their full code by email with correct subject/body usage
