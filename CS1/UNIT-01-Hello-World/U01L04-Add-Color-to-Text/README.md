**# U01L04 – Add Color to Text**  
Overview
In this lesson, students revise their original six‑line story from U01L03 by adding Minecraft color section codes (for example, §6, §c) to their player.say() messages. The focus is on improving readability and visual impact while keeping the story clear, intentional, and easy to follow. Students borrow ideas from a demonstration project and apply color choices thoughtfully rather than randomly.
Estimated Time: 45 minutes

**Focus Skills**

Revising existing code rather than starting over
Using Minecraft color section codes inside strings
Applying visual styling intentionally to enhance meaning
Maintaining correct syntax while editing text
Learning from and adapting ideas seen in a demonstration project


**Evidence of Completion**
Student work should demonstrate:

A revised version of the U01L03 story project
Six or more player.say() commands with color codes added
Use of Minecraft section color codes placed correctly inside the text
Color choices that:

- Improve readability
- Match the mood, speaker, or purpose of each line

Working code that runs without errors
A shared project link submitted as instructed


**Success Criteria (At a Glance)**<br>

✅ Color section codes used correctly<br>
✅ Colors are intentional, not random<br>
